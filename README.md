# MoeensQuiz 
   ### About
   This is a CLI based quiz app.
___

## dependencies
   `nodejs`
   `chalkjs`
   `readlineSync`
   
   
### How to run:
   - Clone this repo
   - cd into the folder
   - run `npm install`
   - run `node main.js`

Viola!
